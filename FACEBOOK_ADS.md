## 1. Campaign Setup in Meta Ads Manager

### Campaign
- **Objective:** Leads (NOT Traffic)
- **Campaign name:** `CWS - SoftwareDev - USA - LeadGen - June 2026`
- **Special ad categories:** None
- **Budget type:** Daily budget — start at **$20/day**

### Ad Set
- **Conversion location:** Website
- **Performance goal:** Maximize number of leads
- **Pixel:** Codewave Studio Pixel
- **Conversion event:** `Lead` (fires when contact form submits — already wired)

### Audience Targeting

**Location:**
- Country: **United States only**
- Optional narrower test: Texas, Florida, California, New York, Arizona (states with high small business density)

**Age:** 28 – 60

**Gender:** All

**Detailed Targeting (Interests + Behaviors — combine these):**

Layer 1 (must match at least one):
- Small business
- Entrepreneurship
- Business owner
- Startup company
- Online business

Layer 2 (must ALSO match at least one — narrows audience to actual owners):
- Job titles: Founder, Owner, CEO, President, Managing Director
- Behaviors: Small business owners (Facebook category)
- Interests: Shopify, Stripe, QuickBooks, Mailchimp, HubSpot, Salesforce

**Exclude:**
- Job titles: Web Developer, Software Engineer, Programmer, UI/UX Designer
- Employers: Google, Meta, Microsoft, Amazon (employees of big tech)
- Interests: Web development, Software development (these are competitors / fellow devs)

**Languages:** English (All)

**Estimated audience size you want to see:** 2M – 10M (narrow enough to be qualified, wide enough to scale)

## 3. Budget & Timeline

### Week 1 (Test phase)
- **Daily budget:** $20/day = ~$140 total
- **Goal:** Get 5+ leads, identify winning ad
- **Decision point:** Cost per lead under $30 = scale. Over $50 = revise creative/audience.

### Week 2–4 (Scale phase)
- **Daily budget:** $35/day on winning ad
- **Goal:** 15–25 leads/week
- **Target:** Convert 1 in 5 leads into a paying client → 3–5 clients/month

### Month 2+ (Profit & expand)
- Reinvest 30% of project revenue
- Scale daily budget to $60–100/day
- Test new audiences (specific states, specific industries)
- Add Google Ads ("custom software developer near me")

## 4. Key Metrics to Watch (in Ads Manager)

| Metric | Good | Bad — Fix It |
|---|---|---|
| CTR (Click-Through Rate) | >1.5% | <0.8% (creative is weak) |
| CPM (Cost per 1000 impressions) | $10–25 | >$40 (audience too narrow) |
| Cost Per Lead | $15–35 | >$50 (landing page or audience issue) |
| Landing page conversion rate | >5% | <2% (improve headline/form) |

## 6. Setup Checklist (do these in order)

1. [ ] Create Facebook **Business Page** for Codewave Studio
2. [ ] Create **Business Manager** at business.facebook.com
3. [ ] Create **Ad Account** (currency: USD, timezone: America/New_York)
4. [ ] Add payment method (international debit card / Payoneer card works)
5. [ ] Go to **Events Manager** → create a **Pixel** → copy the Pixel ID
6. [ ] Paste Pixel ID into `components/MetaPixel.tsx` (replace `YOUR_PIXEL_ID_HERE`)
7. [ ] Deploy site to Vercel: `git push` → import on vercel.com
8. [ ] Buy domain `codewavestudio.com` on Namecheap → point to Vercel
9. [ ] Verify pixel is firing using **Meta Pixel Helper** Chrome extension
10. [ ] Get Web3Forms access key at https://web3forms.com → paste into `app/contact/page.tsx`
11. [ ] Send yourself a test message via the contact form — confirm Gmail receives it
12. [ ] In Events Manager, add `Lead` as a **custom conversion event**
13. [ ] Create the campaign with the targeting above
14. [ ] Upload 3 ad variants
15. [ ] Launch with $20/day budget
16. [ ] Check daily for first 3 days, then every 2–3 days

## 8. Compliance Notes (USA market)

- Ad copy must NOT make unverifiable claims ("guaranteed 10x ROI" gets ads rejected)
- Use real testimonials only — fake reviews violate Meta's Community Standards
- "USA businesses only" wording in ad helps — Meta auto-rejects ads that seem to discriminate, but limiting service area is fine
- Add a basic **Privacy Policy** page on the site before launching ads (Meta requires it) — I can build one if you want
